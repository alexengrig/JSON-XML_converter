Number of decompositions Open on Hyperskill

[ALERT-warning]This problem is a bit complicated and require some additional knowledge and googling. You can just skip it if you are a beginner or don't have enough time. But it is a good case to improve your problem solving and algorithmic skills.[/ALERT]

Read the integer \( N \) (\( 1 \leq N \leq 40 \)) from the standard input and list all the decompositions of NN into a sum of positive integers. The addends should go in non-ascending order within each decomposition.

Output all decompositions in the lexicographical order.

Please, use a recursive method to write your solution.


Sample Input:
5
Sample Output:
1 1 1 1 1
2 1 1 1
2 2 1
3 1 1
3 2
4 1
5



Memory limit: 256 Mb
Time limit: 5s
